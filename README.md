# Установка

1. Скачайте репозиторий (`Code` → `Download ZIP`) и распакуйте архив.
2. Откройте браузер и перейдите на:
   - **Chrome / Edge / Яндекс:** `chrome://extensions/`
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку с файлами расширения (где находится `manifest.json`).
6. Готово! Обновите открытые вкладки (`Ctrl + F5`) или откройте сайт заново.

> После установки расширение автоматически добавляет слово **«Король»** после слова **«Первый»** на поддерживаемых страницах.
